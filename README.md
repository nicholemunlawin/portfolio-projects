# Nichole Munlawin Portfolio

Static portfolio website ready for GitHub upload and GitHub Pages hosting.

## Files

- `index.html` - main homepage for GitHub Pages
- `style.css` - site styling
- `script.js` - scroll and reveal interactions

## Upload to GitHub

1. Create a new repository on GitHub.
2. Upload only `index.html`, `style.css`, `script.js`, `README.md`, and `.gitignore`.
3. Make sure `index.html` stays in the repository root.
4. If you want the site live, enable GitHub Pages in the repository settings.

Do not upload `nicholemunlawin-portfolio.html`. It is a duplicate local copy and is not needed for GitHub Pages.

## GitHub Pages

Use:

- Source: `Deploy from a branch`
- Branch: `main`
- Folder: `/ (root)`

After publishing, GitHub Pages will serve `index.html` automatically.
